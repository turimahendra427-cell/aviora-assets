---

## 🚀 Usage

Google AI Studio or any compatible build system should:

1. Download all asset packs.
2. Extract the archives.
3. Organize assets by category.
4. Generate an asset manifest.
5. Optimize assets for web.
6. Build the AVIORA virtual world.
